# Redux Blog Posts
Simple Blog Posts for Redux with React, React Router, Redux Form and Babel support.

### Getting Started

1. Open project folder:
```
cd blog-posts
```

2. Build dependencies:
```
npm install
```

3. Start local server:
```
npm start
```

4. Open in web browser:
```
http://localhost:8080
```
