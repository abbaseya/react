# Redux Thunk Firebase
Simple Contact List for Redux with React, Redux Thunk and Babel support.

### Getting Started

1. Open project folder:
```
cd firebase-thunk
```

2. Build dependencies:
```
npm install
```

3. Start local server:
```
npm start
```

4. Open in web browser:
```
http://localhost:8080
```
